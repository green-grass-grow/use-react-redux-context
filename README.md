# Minfront

My minimum frontend boilerplate (201811)

- yarn
- parcel
- typescript

## How to dev

```bash
$ git clone git@github.com:mizchi-sandbox/minfront.git your_app_name
$ yarn install
```

- `yarn dev`: Start server on `http://localhost:1234`
- `yarn build`: Generate SPA to `dist`
